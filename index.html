IMAGE SEARCH

Example creation usage:

GET https://image-search-0904.herokuapp.com/api/imagesearch/"search string"

For example

GET https://image-search-0904.herokuapp.com/api/imagesearch/tiger

Get latest searches (default 10):

GET https://image-search-0904.herokuapp.com/api/latest/imagesearch
GET https://image-search-0904.herokuapp.com/api/latest/imagesearch?latest=50
